# Поздравлятор

Веб сайт для ведения списка дней рождений с возможностью просмотра, удаления, добавления, редактирования записей.

## Технологии

#### Backend:

* ASP.Net Web Api
* PostgreSql

#### Frontend:

* Vue
* Quasar
